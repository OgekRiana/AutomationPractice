What we need?

  1. Download dan install Java Software Development Kit (JDK)
  2. Download "Eclipse IDE for Java Developers" disini
  3. Download Selenium Java Client Driver disini

What to do?

Configurasi Eclipse IDE dengan WebDriver

Buatlah sebuah project java, tentukan nama dan lokasi penyimpanan project tersebut. Pada bagian JRE pilih "Use an execution environment JRE". Untuk project layout pilih "Create separate forlders for sources and class file. Klik finish button.

Java project telah berhasil dibuat selanjutnya adalah mengintegrasikan Selenium WebDriver. Klik kanan pada project yang telah dibuat kemudian pilih "Properties". Pada properties dialog pilih "Java Build Path" di sebelah kiri, kemudiah arahkan pada tab "Libraries". Klik "Add External JARs.." kemudian select semua file .jar yang ada pada Selenium Java Client Driver yang sudah diunduh sebelumnya. Klik "Ok" dan Java project dengan Selenium WebDriver siapa digunakan.
